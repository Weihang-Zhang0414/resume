# 基于鸿蒙的外卖软件开发

**项目组长** | 2024年1月 - 2024年6月

**TL;DR:** 基于 DevEco Studio 平台，采用 Java 开发了鸿蒙原生外卖客户端，设计了购物车计算与下单状态机逻辑，实现与远程 MySQL 数据库的网络数据交互。

## 🎯 项目目标
开发一款面向 HarmonyOS 平台的高效原生外卖订餐客户端，包含商品浏览、加购购物车、订单生成及状态同步等核心功能，探索鸿蒙生态下的移动网络请求与状态持久化方案。

## 💻 技术栈
DevEco Studio, Java (HarmonyOS SDK), MySQL 后端数据库, HttpURLConnection 网络组件, 本地模拟器仿真

## 🛠️ 我的核心贡献
* **网络请求与 API 对接:** 独立编写网络通信模块，基于 `HttpURLConnection` 和异步任务（AsyncTask）发送 HTTP 请求，实现鸿蒙客户端与远程后台服务器的 JSON 数据安全交互。
* **购物车核心状态机逻辑:** 编写高可靠的购物车逻辑，实现商品动态加购、价格实时累加计算、多规格属性选定，并结合本地缓存防止页面重启丢失购物车状态。
* **模拟调试与性能调优:** 利用 DevEco 模拟器多维度联调，分析处理内存与网络请求延迟问题，优化列表滑动流畅度与图片异步加载逻辑。

## 🏆 成果与量化影响
* 成功在模拟器端跑通了“商品列表加载 - 购物车计算 - 订单生成 - 后端数据库写入”的闭环业务流程。
* 实现了高响应的加购与结账逻辑，订单提交流程及数据库反馈平均响应时间在 500ms 以内，性能稳定。
