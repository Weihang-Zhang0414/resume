# SenviX 全域智行：视障人群智能自主导航盲杖
**项目负责人** | 2026年3月 - [填写结束时间或至今]

**TL;DR:** 带领团队研发了一款软硬件结合的智能物理盲杖，基于 SLAM 建图与 YOLO 视觉识别技术，在树莓派端侧实现了视障人群的低延迟环境感知与动态避障。

## 🎯 项目目标
针对视障人群面对复杂路况的安全痛点，从0到1研发一款能够实时识别前方高频障碍物、构建局部三维地图并动态规划安全路径的智能辅助硬件设备。

## 💻 技术栈
Python, YOLO 目标检测算法, SLAM 实时建图与定位, 树莓派 (Raspberry Pi), 多传感器数据融合

## 🛠️ 我的核心贡献
* **系统架构与开发统筹:** 设计了涵盖“传感器采集-边缘侧处理-触觉/语音反馈”的完整硬件系统闭环，并主导整体软硬件开发进度。
* **视觉算法端侧部署:** 针对道路常见障碍物（如台阶、悬空物、路障等）微调 YOLO 算法，并成功将其轻量化部署于树莓派平台，保障端侧推理的实时性。
* **空间感知与导航规划:** 运用 SLAM 技术融合多传感器数据实现局部地图构建，为动态路径规划和避障策略提供高精度的空间坐标支撑。

## 🏆 成果与量化影响
* 成功构建并落地可独立稳定运行的物理原型机。
