# 基于 Arduino 的智能眼控轮椅项目

**项目负责人** | 2024年7月 - 2024年8月

**TL;DR:** 研发了一款基于人眼视线追踪的辅助出行硬件设备，利用 Matlab 图像处理算法识别瞳孔朝向，并转换为控制信号通过蓝牙控制 Arduino 轮椅驱动系统。

## 🎯 项目目标
面向行动不便的重度残障群体，开发一套能通过眼球运动实现行驶方向控制的智能轮椅控制系统，提供稳定、灵敏且具备避障保护的自主助行解决方案。

## 💻 技术栈
Matlab, Viola-Jones 人脸检测算法, 圆形霍夫变换 (Circle Hough Transform), 蓝牙通信, Arduino, 嵌入式 C/C++, 传感器数据融合

## 🛠️ 我的核心贡献
* **眼动识别与瞳孔定位:** 基于 Matlab 平台，应用 Viola-Jones 算法完成人脸及眼部实时区域检测。针对图像进行自适应裁剪和灰度化处理，优化帧率。引入霍夫变换（Hough Transform）进行高精度圆形检测，精准锁定瞳孔中心坐标及视线偏转方向。
* **信号映射与自定义控制协议:** 设计了一套串口协议，将图像中瞳孔偏离中心的像素偏移量转化为特定的运动方向和速度控制指令。利用蓝牙模块实现 Matlab 决策端与轮椅底盘硬件端的低延迟无线传输。
* **底盘硬件驱动与逻辑开发:** 编写 Arduino 核心控制器代码，驱动轮椅电机差速旋转。集成超声波传感器，实现检测到前方障碍物时自动刹车避障的安全冗余逻辑。

## 🏆 成果与量化影响
* 成功搭建出智能眼控轮椅的软硬件原型系统，实测眼球转向信号到轮椅电机响应的闭环延迟在 200ms 以内。
* 实现了姿态控制指令自定义化与控制动作标准化，系统对上/下/左/右视线偏移的识别准确率高，运行稳定。
* 项目获得导师的高度评价，并获得英国伦敦布鲁内尔大学国际夏校导师的**推荐信**。
