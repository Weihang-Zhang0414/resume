# 基于鸿蒙的个人博客软件开发

**项目组长** | 2024年8月 - 2025年1月

**TL;DR:** 使用 DevEco Studio 开发了一款运行在 HarmonyOS 上的原生博客应用，基于 Java 编写前端页面逻辑，使用 SQLite 数据库建立轻量级本地存储。

## 🎯 项目目标
开发一款面向 HarmonyOS 用户的原生个人博客客户端，支持文章浏览、本地草稿保存、分类检索与用户个人信息管理，提升用户在鸿蒙生态下的移动端阅读与写作体验。

## 💻 技术栈
DevEco Studio, Java (HarmonyOS SDK), SQLite 本地数据库, HarmonyOS 原生 UI 组件, 本地模拟器仿真

## 🛠️ 我的核心贡献
* **原生 UI 设计与页面构建:** 使用 HarmonyOS 自有的布局与 UI 控件（包括 Ability、Slice、ListContainer 等），开发响应式界面，实现文章列表、详情展示及分类过滤等前端交互逻辑。
* **本地数据库架构设计:** 设计 SQLite 数据库架构，编写本地 SQL 语句，建立博客文章、标签及草稿箱的本地关系型存储表，实现高效的本地增删改查（CRUD）操作。
* **应用生命周期与路由管理:** 管理 HarmonyOS Page Ability 及其 Slice 的生命周期，确保页面切换与数据传递的流畅性，避免内存泄露与切换卡顿。

## 🏆 成果与量化影响
* 成功在 DevEco Studio 本地模拟器上完成了个人博客软件的完整功能调试，所有业务流运行流畅。
* 实现了高响应度的本地数据检索，草稿保存与文章加载均在毫秒级内完成，提供流畅的原生操作体验。
