# HarmonyOS-based Personal Blog Software

**Team Leader** | August 2024 - January 2025

**TL;DR:** Developed a native HarmonyOS personal blog application using DevEco Studio and Java, integrating an SQLite database for high-performance offline data storage.

## 🎯 Project Objectives
To build a native mobile blogging application for the HarmonyOS ecosystem, providing features such as article browsing, draft editing, tag filtering, and local SQLite data caching for a smooth user experience.

## 💻 Tech Stack
DevEco Studio, Java (HarmonyOS SDK), SQLite Database, HarmonyOS Native UI Components, DevEco Local Emulator

## 🛠️ Core Contributions
* **Native UI/UX Development:** Implemented frontend layouts utilizing HarmonyOS SDK layout systems (such as Ability, Slice, and ListContainer). Engineered responsive views for article feed, detailed reading, and categorization.
* **SQLite Database Design:** Designed relational database schemas for local storage of articles, drafts, and user configurations. Formulated SQL commands to perform fast local CRUD (Create, Read, Update, Delete) operations.
* **Lifecycle & Routing Management:** Controlled page navigation paths and coordinated Page Ability slice lifecycles to maintain low memory consumption and prevent UI freezing.

## 🏆 Key Results & Quantified Impact
* Successfully built and debugged the functional personal blog application prototype on the DevEco Studio local emulator.
* Achieved sub-second query and save latency for local blog drafts, providing users with a responsive offline writing environment.
