# 2025年美国大学生数学建模竞赛 (MCM) - C题 (奥运奖牌预测)

**核心团队成员** | 2025年

**TL;DR:** 针对2028年洛杉矶奥运会奖牌榜预测问题，综合运用时间序列分析、机器学习和因果推断方法构建了多维度预测与评估模型 。

## 🎯 项目目标

基于历届夏季奥运会的历史奖牌数据、参赛运动员信息及赛事项目分布，预测2028年洛杉矶奥运会各国的奖牌获取情况 。同时，评估主办国效应、特定赛事类型以及“明星教练”效应对奖牌分布的实质性影响 。

## 💻 技术栈

Python, ARIMA, GM(1,1) 灰色预测, 逻辑回归 (Logistic Regression), 随机森林 (Random Forest), 贝叶斯回归 (Bayesian Ridge), 层次分析法 (AHP), 双重差分模型 (DID) 

## 🛠️ 我的核心贡献

* 
**时序数据挖掘与预测:** 针对拥有充足历史数据的国家使用 ARIMA 模型计算年均增长率进行预测；针对数据不连续的国家，独立构建了 GM(1,1) 灰色预测模型，解决数据稀疏性问题 。


* 
**零突破概率评估:** 构建以交叉熵为损失函数的逻辑回归模型，结合最大似然估计，对77个尚未获得过奖牌的国家在2028年实现“零的突破”的概率进行了量化预测 。


* 
**多因素回归与机器学习:** 提取运动员数量和赛事数量作为特征矩阵，对比了线性回归、随机森林和贝叶斯回归在预测具体金银铜牌数量上的表现，并进行了特征标准化处理 。


* 
**因果推断与政策量化:** 创新性地结合层次分析法 (AHP) 和双重差分模型 (DID)，量化了“明星教练”对国家队成绩的净效应，并为日本、法国、英国的特定弱势项目提出了“教练投资”的量化建议 。



## 🏆 成果与量化影响

* 成功输出了包含置信区间的2028年洛杉矶奥运会奖牌榜预测完整名单，并精准定位了预测概率排名前十的“潜在首枚奖牌”获得国 。


* 通过普通最小二乘法 (OLS) 得出量化结论：主办国身份对奖牌分布有显著正向影响，平均能带来约 2.69 枚额外奖牌 。


* 运用 MAE、MSE 和决定系数 ($R^2$) 对模型进行了交叉验证，确保了多元回归模型在未知数据上的泛化能力与预测稳定性 。
