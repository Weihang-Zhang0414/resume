# 赤道仪模型设计

**外援 / 机械建模工程师** | 2024年5月 - 2025年6月

**TL;DR:** 协助完成了赤道仪模型的高精度 3D 机械建模，绘制了全套的标准工程制图与零件装配图，保障了实物加工的高精度对接。

## 🎯 项目目标
设计一款结构合理、传动精确、符合天文跟踪观测标准的赤道仪模型，通过标准化的工程图纸为后续的物理样机加工与组装提供精确的工艺规范。

## 💻 技术栈
SolidWorks, Auto CAD, 3D 机械建模, 工程制图, 机械零件设计, 机械装配设计

## 🛠️ 我的核心贡献
* **高精度 3D 零件建模:** 使用 SolidWorks 完成赤道仪核心结构（包括赤纬轴、赤经轴、蜗轮蜗杆传动机构、重锤架及三脚架连接件）的三维参数化建模。
* **工程图与装配图绘制:** 绘制符合国家机械制图标准的 2D 零件图和装配工程图，标注详尽的尺寸公差、形位公差及表面粗糙度要求。
* **运动仿真与结构干涉检查:** 运行 SolidWorks Motion 插件进行多轴运动仿真，检查各部件在极限偏转角度下的机械干涉情况，并根据仿真结果优化了齿轮副间隙与传动轴结构。

## 🏆 成果与量化影响
* 成功交付了包含数十个精密零件的完整赤道仪三维装配模型与全套 2D 工程图纸。
* 通过精细化三维建模与尺寸链计算，确保加工图纸公差分配科学，使样机零件加工一次装配成功率达 100%。
* 项目成果在机械结构设计训练及学校竞赛中获得了优秀反馈。
