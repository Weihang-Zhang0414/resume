# 基于机器视觉的生活垃圾智能分类系统
**项目负责人 / 核心开发者** | 202X年X月 - 202X年X月

**TL;DR:** 研发并落地了一款软硬件结合的智能分类垃圾桶，通过在边缘设备部署深度学习视觉模型，实现对生活垃圾的自动精准识别与自动化机械分拣。

## 🎯 项目目标
针对传统垃圾分类中人为判断错误率高、效率低下的痛点，打造一套涵盖“视觉感知-边缘推理-机械控制”的完整端到端智能环保物联网设备。

## 💻 技术栈
Python, 计算机视觉 (CNN / OpenCV), 边缘计算 (Raspberry Pi / 嵌入式平台), 传感器融合, 舵机与多线程控制

## 🛠️ 我的核心贡献
* **视觉算法端侧部署:** 采集并预处理多类别生活垃圾图像数据，训练图像分类/目标检测模型，并将其轻量化部署至边缘计算节点，保障端侧推理的实时性与脱机运行能力。
* **软硬件协同与控制:** 编写底层控制逻辑，将视觉模型的推理结果映射为具体的硬件指令，驱动舵机和连杆机构，实现四分类舱门的精准自动开合。
* **全链路系统集成:** 整合摄像头视频流获取、AI 模型推理、屏幕交互界面展示与机械结构动作，攻克了系统并发处理的瓶颈，构建了高鲁棒性的智能硬件平台。

## 🏆 成果与量化影响
* 成功搭建具备屏幕交互与自动化分拣功能的四分类物理样机，系统可长期稳定运行。
* 优化后的模型在端侧对常见生活垃圾的平均识别准确率达到 **[填写准确率，如 92%]**。
