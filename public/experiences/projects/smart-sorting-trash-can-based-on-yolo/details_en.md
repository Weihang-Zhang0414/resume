# Machine Vision-Based Smart Domestic Waste Sorting System
**Project Lead / Core Developer** | [Month, Year] - [Month, Year]

**TL;DR:** Developed and deployed a hardware-software integrated smart trash bin, utilizing deep learning vision models on edge computing devices to achieve automatic, high-precision waste recognition and mechanical sorting.

## 🎯 Objective
To address the high error rates and low efficiency of manual waste sorting by building an end-to-end smart IoT environmental device that integrates "visual perception - edge inference - mechanical control."

## 💻 Tech Stack
Python, Computer Vision (CNN / OpenCV), Edge Computing (Raspberry Pi / Embedded Platforms), Sensor Fusion, Servo & Multi-threading Control

## 🛠️ Core Contributions
* **Edge-Side Vision Algorithm Deployment:** Collected and pre-processed multi-class domestic waste image data, trained image classification/object detection models, and deployed lightweight versions to edge computing nodes, ensuring real-time inference and offline execution capabilities.
* **Hardware-Software Integration:** Developed the low-level control logic to map visual model inference results into specific hardware instructions, driving servos and linkage mechanisms for the precise, automatic opening of the four-compartment bins.
* **Full-Stack System Integration:** Integrated camera video streams, AI model inference, screen UI displays, and mechanical actions. Overcame system concurrency bottlenecks to build a highly robust smart hardware platform.
