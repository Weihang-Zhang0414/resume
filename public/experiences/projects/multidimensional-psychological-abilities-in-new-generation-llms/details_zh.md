# 新一代大模型多维心理学能力研究

**论文第一作者** | 2025年7月 - 2025年8月

**TL;DR:** 聚焦人工智能与认知科学交叉领域，系统评估了新一代大模型（如 ChatGPT-5 和 Gemini 2.5）在心理理论（Theory of Mind）及认知偏差上的表现，并作为第一作者发表了被 EI 检索的学术论文。

## 🎯 项目目标
利用经典心理学范式（如 Kosinski 错误信念任务框架），对大语言模型（LLMs）的类人认知能力和心理学属性进行多维度量化评估，探索大模型在复杂心智和社会认知任务中的能力边界与潜在缺陷。

## 💻 技术栈
Python, ChatGPT-5, Gemini 2.5, 大语言模型 (LLM), 心理理论 (Theory of Mind), 统计分析 (Pearson 相关分析 / Bland-Altman 分析), Pandas, NumPy, SciPy

## 🛠️ 我的核心贡献
* **实验设计与复现:** 深入探索大模型在认知心理学层面的类人属性，基于经典心理理论范式设计并编写自动化评测脚本，系统化复现了 Kosinski 错误信念任务评测框架。
* **海量实验数据挖掘:** 独立处理并解析大模型生成的大规模 JSON 响应数据，使用 Python（Pandas、NumPy）完成特征提取、数据清洗与结构化建模。
* **统计学评估与推断:** 引入 Pearson 相关系数、Bland-Altman 偏差分析等严谨统计学工具，评估模型在不同提示词模板与上下文长度下的决策一致性及偏差分布，绘制高质量学术图表。
* **学术论文撰写与录用:** 独立完成英文学术论文撰写，包括引言、方法论、实验设计、数据分析及结论。

## 🏆 成果与量化影响
* 作为第一作者撰写的学术论文被第五届电子信息工程与数据处理国际会议（EIEPD 2026）录用，支持 **EI Compendex** 与 **Scopus** 双检索。
* 成功建立了一套针对 LLM 认知能力的标准化评估 pipeline，实现了对模型心智理论水平的自动化量化分析。
